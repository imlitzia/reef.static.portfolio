# Aarif Suksai Portfolio

A static portfolio website designed with a dark black-red-white aesthetic and Pinterest-style masonry layout.

## Run locally

Open `index.html` directly in a browser.

## Deploy to GitHub Pages

1. Create a repository on GitHub.
2. Push these files to the default branch (for example `main`).
3. Go to repository `Settings` -> `Pages`.
4. Under **Build and deployment**, choose:
   - **Source**: Deploy from a branch
   - **Branch**: `main` and `/root`
5. Save and wait for GitHub Pages to publish your site.

Your site URL will be:

`https://<your-github-username>.github.io/<repository-name>/`
